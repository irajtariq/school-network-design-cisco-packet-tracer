# VLAN-Based School Network Design

A Cisco Packet Tracer simulation of a VLAN-based school network developed as a Computer Networks course project. The network is designed to improve security, scalability, and communication by segmenting departments into separate Virtual LANs (VLANs) while enabling inter-VLAN communication.

---

## Project Objectives

- Design a structured school network
- Implement VLAN segmentation
- Configure inter-VLAN routing
- Improve network security and organization
- Simulate the network using Cisco Packet Tracer

---

## Features

- Multiple VLANs for different departments
- Router-on-a-Stick configuration
- Trunk links between switches
- IP Addressing
- Inter-VLAN Communication
- Scalable network design

---
